# Streamlit Demo

Готовый чат-интерфейс для демонстрации вашего AI-решения на питче.

## Быстрый старт

```bash
cd templates/streamlit-demo
pip install -r requirements.txt
cp .env.example .env   # заполните API-ключи
streamlit run app.py
```

## Что внутри

- Чат-интерфейс с историей сообщений
- Сайдбар с выбором провайдера и модели
- Заглушка для замены на ваш реальный бэкенд

## Зачем это нужно

На питче жюри хочет видеть **работающий продукт**. Streamlit позволяет за 30 минут собрать UI, который выглядит как настоящее приложение.

## Советы

- Добавьте загрузку файлов через `st.file_uploader()` если ваш кейс про документы
- Используйте `st.columns()` для дашбордов и аналитики
- `st.metric()` отлично показывает KPI и экономический эффект
